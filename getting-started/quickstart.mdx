### Step 1: Verify your Dealflow account

<AccordionGroup>
  <Accordion icon="play" title="Set up your account">
    1. Sign up to Dealflow and get accepted from the waitlist 
    2. Complete the account setup flow as described in the webapp
  </Accordion>
  <Accordion icon="shield-check" title="Get verified">
    1. This usually takes 0 - 1 days
    2. Continue to step 2 once you're verified
    3. Upload more documentation if you don't get verified
    
    <Tip>Feel free to try again if you get your verification declined. We do take second looks.</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Get your API key

<AccordionGroup>
  <Accordion icon="radio" title="Enable the API plugin">
    1. Navigate to the plugin tab in the webapp
    2. Enable the API plugin if it's not already enabled
</Accordion>
<Accordion icon="key" title="Generate your API key">
    1. Go to the API tab in the company settings page 
    2. Generate the key 

    <Tip>Check out the authentication tab if you want more details on this step!</Tip>
  </Accordion>
</AccordionGroup>
